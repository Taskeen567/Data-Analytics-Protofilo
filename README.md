# Pizza Sales Dashboard (Excel)

## 📊 Overview

This project is an interactive Pizza Sales Dashboard built using Power BI to analyze sales performance and customer ordering patterns.

## 🎯 Objective

To analyze sales data and identify trends in revenue, order volume, and product performance.

## 📁 Dataset

The dataset includes:

* Order ID
* Pizza Category
* Quantity
* Order Date
* Order Date
*  Order Time
* Pizza Name
* Pizza Size
* Unit Price
* Total Price


## 🧹 Data Cleaning

* Removed duplicates
* Verified that no missing values were present
* Checked and standardized date and time formats
* Created a Date Table for time-based analysis
* Created calculated columns to classify time into Morning, Afternoon, and Night shifts
* Validated sales, quantity, and price values
* Multiple Nulls are removed during the data cleaning process
* Created DAX measures for key KPIs
* Ensured data consistency across all tables
* Handled missing values

## 📊 Analysis & Dashboard

* Created KPIs like total revenue, total orders, and average order value
* Used Calculated Columnns And Measures
* Built charts for sales trends and category performance
* Added slicers for filtering

## 📌 Key Insights

* Identified top-selling pizza categories
* Analyzed peak sales periods
* Observed customer ordering behavior

## 💼 Business Impact

* Helps improve sales strategy
* Identifies best-performing products
* Supports better inventory planning

## 🛠 Tools Used

* Power BI
* DAx
* Power Query
