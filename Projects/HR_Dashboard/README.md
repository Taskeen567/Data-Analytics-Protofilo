# HR Insights Dashboard (Excel)

## 📊 Overview

This project is an interactive HR dashboard built in Microsoft Excel to analyze employee data and provide meaningful insights for decision-making.

## 🎯 Objective

The main objective of this dashboard is to monitor workforce trends such as employee distribution, attrition rate, and diversity.

## 📁 Dataset

The dataset includes:

* Employee ID
* Department
* Job Role
* Salary
* Gender
* Tenure

## 🧹 Data Cleaning

* Removed duplicate records
* Handled missing values
* Standardized data formats using Power Query

## 📊 Analysis & Dashboard

* Created Pivot Tables for KPI calculations
* Built charts for visualization
* Added slicers for interactivity

## 📌 Key Insights

* Identified departments with high attrition
* Analyzed gender distribution across roles
* Observed employee tenure trends

## 💼 Business Impact

This dashboard helps HR teams:

* Improve employee retention
* Make better hiring decisions
* Understand workforce patterns

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables & Charts
* Power Query
