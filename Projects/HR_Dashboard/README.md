
## 📊 Overview

This project is an interactive HR dashboard built in Microsoft Excel to analyze employee data and provide meaningful insights for decision-making.

## 🎯 Objective

The main objective of this dashboard is to monitor workforce trends such as employee distribution, attrition rate, and diversity.

## 📁 Dataset

The dataset includes:

* Employee ID
*Department
*Salary
*Performance Score
*Termination Date
*Gender
*Position
*Recruitment Source



## 🧹 Data Cleaning

* Removed duplicate records
* Fixed incorrect date formats
* Handled missing values in date fields by creating a helper column and replacing missing values with 0 to ensure data consistency.
* Identified blank Manager IDs and handled them by creating a helper column and replacing missing values with 0 to ensure data consistency.
* Verified salary and performance data
* Created calculated columns
* Created  measures for KPIs
* This improved data accuracy and reliability

## 📊 Analysis & Dashboard

* Created Pivot Tables for KPI calculations
* Built charts for visualization
* Added slicers for interactivity

## 📌 Key Insights

* Identified Performance monitoring
* Analyzed gender distribution across roles
* Observed employee tenure trends

## 💼 Business Impact

This dashboard helps HR teams:

* Improve employee retention
* Make better hiring decisions
* Understand workforce patterns

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables & Charts
* Power Query
